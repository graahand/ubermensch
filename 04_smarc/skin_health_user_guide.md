
This instructions is for Derma Care (skin health analysis project) after the setup is  completed. 

## User Guide

1. Use the command [docker-compose up --build] which starts the derma-care application. 
2. After the execution of the above command, open the URL [127.0.0.1::8002] which opens up the skin health analysis interface. 
3. There are two options for analyzing the skin's health, either upload existing image from the computer or capture the image in real time using [Use Camera] button. 
4. Once you upload the image of your face to the application, after 15-20 seconds, the results will be shown. For the first time execution  it might take a bit more time. 
5. The results after the image is uploaded to the system are (skin's age, primary skin condition[dark circles, pigmentation, normal] and face's shape) along with detailed analysis which consist of the analysis results and skincare recommendations according to the detected face's health.
